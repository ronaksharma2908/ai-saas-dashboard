# 🚀 AI SaaS Dashboard

An interactive **AI SaaS Dashboard** built with **React + Vite** (frontend) and **Flask** (backend).  
It provides real‑time monitoring of AI endpoints, KPIs, alerts, and visualizations in a clean, responsive UI.

---

## ✨ Features
- 📊 KPI Cards – Uptime, Requests, Errors
- 🍩 Donut Chart – Request distribution
- 📈 Spark Bar – Trends visualization
- 🔌 Endpoint Table – API status & latency
- ⚠️ Alert Feed – Real‑time alerts
- 🎨 Responsive design – Desktop & mobile ready
